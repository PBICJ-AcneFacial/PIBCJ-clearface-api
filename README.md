## Description

ClearFaceAPI is the source backend of clearFace app. A fullstack aplication for acne classification and detection using Machine Learning Methods

## Project setup

```bash
$ npm install
```

Setup Docker Container
```bash
$ docker compose up
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
